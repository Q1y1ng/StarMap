## 项目创新点

### 1. 多页图像联合解析

> 区别于传统逐张 OCR 识别方案，本系统将所有考试页面一次性送入视觉大模型，利用模型的上下文理解能力建立跨页面关联。

**技术路线图**：
```
试卷扫描件（多页PNG）
    ↓
一次性送入 Doubao Vision API
    ↓
模型建立跨图片上下文
    ↓
结构化 Markdown 输出（含题目-答案-分数对应关系）
```

### 2. 纯规则引擎学情诊断

> 学情分析不依赖大模型，基于结构化数据进行规则引擎推理，确保分析结果的可解释性和一致性。

**诊断维度**：
- 综合掌握率计算
- 四维学习画像（掌握率、稳定性、薄弱环节、提升潜力）
- 线性回归风险预警
- 7 天个性化学习计划

### 3. 全自动处理流程

> 从上传到分析报告生成，全程无需人工干预。

**处理流程**：
```
上传 → 试卷分类 → 视觉识别 → 结构化提取 → 学情分析 → 报告生成
（0干预）（自动）   （LLM）      （规则引擎）    （规则引擎） （模板）
```
